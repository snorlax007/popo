## Quick start

```bash
cd app/
pip install -r requirements.txt
DATABASE_URL=sqlite+aiosqlite:///./popo.db uvicorn app.main:app --reload --port 8000
# → http://localhost:8000 (admin: admin@popo.local / changeme123)
```
